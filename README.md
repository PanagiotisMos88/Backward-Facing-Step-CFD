# Backward-Facing-Step-CFD
This repository contains the files for the CFD study of a turbulent flow over a backward-facing step using OpenFOAM and STAR-CCM+.

## Contents
- `0/` : Initial and boundary conditions for OpenFOAM simulation
- `constant/` : Physical properties, turbulence models, and mesh
- `system/` : Solver controls and numerical schemes
- `report/` : PDF and LaTeX source of the report

## Objective
- Compare OpenFOAM and STAR-CCM+ predictions with experimental data provided by Seegmiller and Driver[1]
- Study velocity profiles, skin-friction coefficient, pressure distribution, and Reynolds shear stress

References
[1]David M Driver and H Lee Seegmiller. Features of a reattaching turbulent shear layer
in divergent channelflow. AIAA journal, 23(2):163–171, 1985.
